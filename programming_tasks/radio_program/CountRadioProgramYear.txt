Программа выполняет подсчет количества действующих радиопрограмм
по годам в countyearsRadio. 1920 год - первая радиопередача соответсвует 0-ому индексу.

Строка strQuery содержит цепочку, состоющую из можества групп из 7 символов, в которых:
первые 4 числа - год начала радиопрограммы, "-" - отделитель, следующее число - количество лет,
которые существовала радиопрограмма, затем отделитель "n".

Строка strQueryYears - содержит чепочку групп символов, состоящих по 4 - год начала радиопрограммы,
которая существует по сей день.
